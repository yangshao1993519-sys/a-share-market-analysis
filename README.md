# A股智洞 | 专业A股市场洞察平台

**全新大众化版本** - 从个人交易复盘工具升级为面向广大投资者的专业市场洞察平台。

## 全新网站架构

- **现代专业设计**：Tailwind CSS + 响应式布局 + 深色金融主题
- **首页**：Hero区 + 实时市场快照 + 今日洞察 + 智能工具 + 学习中心
- **核心功能**：
  - 每日洞察（结构化、专业但易懂）
  - 智能工具箱（筛选器、仓位计算器、情绪指数）
  - 专业复盘模板
  - 学习中心（新手到进阶）
