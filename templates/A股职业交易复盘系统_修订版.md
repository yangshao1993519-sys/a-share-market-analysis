# A股职业交易复盘系统（修订版）

> 基于原版框架，补充板块梯队、题材生命周期、资金性质、政策面、集中度风险、次日作战计划六大模块，调整评分权重，优化量价结构。

---

## 一、核心理念

市场真正重要的，不是指数涨跌。

而是：
- 风险是在扩散还是收缩
- 资金是在进攻还是防守
- 市场有没有赚钱效应
- 主线是否仍具持续性
- 流动性是否正在恶化

职业交易员的复盘，本质是：

**"判断当前市场所处的周期位置，以及未来3-5个交易日的风险收益比。"**

---

## 二、复盘总框架

| 模块 | 核心目的 |

## 三、政策与消息面（新增模块）

> 复盘的第一个动作：扫一圈，有没有足以推翻技术判断的外部变量。

### 检查清单

| 层级 | 关注内容 | 影响等级 |
|---|---|---|
| 国务院/政治局 | 经济定调、资本市场相关表述 | ★★★★★ |
| 证监会 | 交易制度、IPO节奏、监管风向 | ★★★★ |
| 央行 | 降准降息、流动性工具 | ★★★★ |
| 行业政策 | 产业补贴/限制/整顿 | ★★★ |
| 外围市场 | 美股大幅波动（±2%）、恒生异动 | ★★★ |
| 地缘事件 | 军事冲突、制裁 | ★★ |

### 结论模板

> 今日是否存在足以改变市场节奏的政策/事件冲击？【是/否】
> 若存在，其影响方向为：【利好/利空/中性】，预计影响周期：【短期1-2日 / 中期1-2周 / 长期】

## 四、市场环境复盘

### 1、指数环境

| 指标 | 量化标准 | 含义 |
|---|---|---|
| 上证指数涨跌幅 ≥ +1.5% | 强势 | 权重大盘进攻 |
| 上证指数涨跌幅 ≤ -1.5% | 弱势 | 权重风险释放 |
| 创业板 ≥ +2% | 风险偏好提升 | 小票成长增强 |
| 创业板 ≤ -2% | 高弹性方向退潮 | 风险偏好下降 |
| 科创50 ≥ +2% | 科技主线强化 | 资金进攻科技 |
| 两市成交额 ≥ 1.2万亿 | 市场活跃 | 增量资金入场 |
| 两市成交额 ≤ 8000亿 | 市场低迷 | 资金观望 |
| 成交额变化率 ≥ 15% | 放量进攻 | 市场增量 |
| 成交额变化率 ≤ -15% | 缩量退潮 | 风险收缩 |
| 北向资金 ≥ +50亿 | 外资做多 | 风险偏好增强 |
| 北向资金 ≤ -50亿 | 外资撤退 | 风险偏好下降 |

### 2、分时量能结构（新增）

| 指标 | 计算方法 | 市场含义 |
|---|---|---|
| 早盘30分钟成交占比 | 开盘30分钟成交 ÷ 全日成交 | ≥20% 有资金抢筹，≤12% 交投冷淡 |
| 尾盘30分钟异动 | 最后30分钟成交 ÷ 全日成交 | ≥18% 且指数拉升=抢筹，≥18% 且指数跳水=恐慌出逃 |
| 日内量能分布 | 上午成交 vs 下午成交 | 上午放量下午缩量=持续性存疑 |

### 3、大小盘风格（新增）

| 指标 | 量化标准 | 含义 |
|---|---|---|
| 中证2000/沪深300比值上升 | ≥+1% | 小票风格占优，情绪活跃 |
| 中证2000/沪深300比值下降 | ≤-1% | 大票护盘，小票失血 |
| 比值连续3日同向 | 趋势确认 | 风格切换信号 |

## 七、资金结构与性质（大幅增强）

### 1、北向资金

| 指标 | 量化标准 | 含义 |
|---|---|---|
| 北向资金 ≥ +50亿 | 外资做多 | 风险偏好增强 |
| 北向资金 ≤ -50亿 | 外资撤退 | 风险偏好下降 |
| 连续3日净流入/出 | 趋势确认 | 方向信号增强 |

### 2、龙虎榜资金性质（新增——关键模块）

> A股的钱分三种，风格完全不同，必须分类统计。

#### 机构席位

| 指标 | 观察方法 |
|---|---|
| 机构净买入总额 | 机构专用席位合计净买入金额 |
| 机构主攻方向 | 买入集中在哪些板块？ |
| 机构 vs 游资分歧 | 机构卖出、游资买入 = 高位分歧，风险信号 |

**机构操作风格**：趋势推进、低吸不追高、持仓周期较长

#### 一线游资

| 指标 | 观察方法 |
|---|---|
| 活跃游资数量 | 今日龙虎榜上榜的知名游资席位数量 |
| 游资进攻方向 | 集中在哪些个股/板块？ |
| 游资仓位信号 | 游资单票买入额是否放大？（放大=积极，缩小=收缩） |

**游资操作风格**：连板接力、快进快出、次日溢价是关键

#### 量化资金

| 指标 | 观察方法 |
|---|---|
| 量化席位活跃度 | 华鑫上海分公司、中金财富等量化席位上榜频率 |
| 量化主导的板块 | 量化主导的板块日内波动剧烈，追高易被收割 |

**量化操作风格**：日内高频、制造流动性、趋势跟随但不接力

### 3、资金性质结论模板（新增）

> 今日市场主导资金为：【机构主导 / 游资主导 / 量化主导 / 三足鼎立】
> 主导资金与当前市场阶段是否匹配？【匹配/不匹配，说明】
> （例：退潮期游资仍活跃 → 不匹配 → 诱多嫌疑，谨慎）

## 八、主线强度与板块梯队（大幅增强）

### 1、主线持续性

| 持续时间 | 主线强度 |
|---|---|
| ≥5天 | 核心主线 |
| 3-5天 | 轮动主线 |
| ≤2天 | 无主线 |

### 2、板块强度评分

公式：板块强度 = 涨停数×2 + 涨幅超5%个股数 - 跌幅超5%个股数

| 分值 | 强度 |
|---|---|
| ≥30 | 极强 |
| 15-30 | 强 |
| 5-15 | 一般 |
| ≤5 | 弱 |

### 3、主线集中度

统计：成交额前20股票中，同一题材占比。

| 占比 | 市场结构 |
|---|---|
| ≥50% | 核心主线 |
| 30%-50% | 多主线 |
| ≤30% | 混沌轮动 |

### 4、板块内部梯队结构（新增——关键模块）

> 每个主线板块必须拆解为四层梯队，判断板块健康度。

```
第一梯队：龙头（市场辨识度最高，引领板块方向）
    ↓
第二梯队：核心跟风（龙头上板后跟进，次日有溢价）
    ↓
第三梯队：补涨（低位挖掘，板块热度扩散的标志）
    ↓
第四梯队：卡位（试图替代龙头，通常在龙头分歧时出现）
```

#### 梯队健康度判断

| 梯队状态 | 含义 | 操作建议 |
|---|---|---|
| 四层梯队完整 | 板块处于主升期，结构健康 | 可积极参与 |
| 龙头独舞，跟风掉队 | 板块进入末升段，随时退潮 | 只持仓不新开 |
| 龙头补跌，补涨上位 | 高低切换，板块内部轮动 | 关注新方向 |
| 梯队全散，无辨识度 | 板块退潮确认 | 清仓回避 |

### 5、主线板块梯队记录表（新增）

| 板块 | 龙头（连板数） | 核心跟风 | 补涨 | 梯队状态 | 操作建议 |
|---|---|---|---|---|---|
| AI | XX（5板） | XX、XX | XX | 梯队完整 | 可参与 |
| 新能源 | XX（3板） | 无明确跟风 | 无 | 龙头独舞 | 观望 |

## 九、题材生命周期定位（新增模块）

> 每一个题材都有生老病死。光数天数没用，要判断它处于哪个阶段。

### 生命周期五阶段

| 阶段 | 特征 | 操作策略 |
|---|---|---|
| **萌芽期** | 首板零星出现，无板块效应，市场关注度低 | 轻仓试错，不上仓位 |
| **发酵期** | 龙头打出辨识度，跟风开始跟进，分歧转一致 | 确认后加仓，主仓位建立期 |
| **高潮期** | 板块涨停潮，媒体大量报道，散户涌入 | 持仓不动或减仓，不开新仓 |
| **分化期** | 后排掉队，龙头高位震荡，板块内涨跌分化 | 只留龙头，清掉跟风 |
| **退潮期** | 龙头补跌，板块集体回落，无新首板出现 | 清仓回避，不抄底 |

### 当前各主线题材生命周期定位

| 题材 | 启动日期 | 当前阶段 | 所处阶段判断依据 | 距退潮预估天数 |
|---|---|---|---|---|
| 题材A | XX月XX日 | 高潮期 | 后排加速，散户热议 | 1-3天 |
| 题材B | XX月XX日 | 发酵期 | 龙头分歧转一致 | 3-5天 |

### 生命周期核心规则

1. **高潮期不买。** 高潮期是卖点，不是买点。
2. **冰点看新题材。** 老主线退潮时，第一个冒出来的新方向值得重点跟踪。
3. **发酵期是最好的上车窗口。** 龙头分歧日如果次日转一致，就是最佳买点。

## 十、集中度风险（新增模块）

> 资金过度拥挤本身，就是最大的风险。

### 1、板块成交集中度

统计：单一板块成交额 ÷ 全市场成交额 ×100%

| 占比 | 风险等级 | 含义 |
|---|---|---|
| ≥30% | ★★★ 高危 | 资金极度拥挤，一旦松动即踩踏 |
| 20%-30% | ★★ 警惕 | 过热，需要关注退潮信号 |
| 10%-20% | ★ 正常 | 健康的主线占位 |
| ≤10% | 分散 | 无集中度风险 |

### 2、个股成交集中度

统计：全市场成交额前10个股合计成交 ÷ 全市场成交额

| 占比 | 风险等级 |
|---|---|
| ≥15% | 过度集中，市场脆弱 |
| 8%-15% | 正常 |
| ≤8% | 资金分散 |

## 十三、市场情绪评分系统（修订权重）

> 权重调整逻辑：量在价先，成交额是市场唯一真实的语言；主线是市场的灵魂。原版情绪权重过高（60分），修订后更均衡。

| 项目 | 原权重 | 修订权重 | 修订理由 |
|---|---|---|---|
| 成交额 | 15 | **25** | 量在价先，一切的前提 |
| 炸板率 + 高位亏钱率 | 20（仅炸板率） | **20** | 退潮先行指标，合并高位亏钱效应 |
| 主线持续性 + 集中度 | 10 | **20** | 市场有没有灵魂 |
| 跌停数量 | 20 | **15** | 风险底线 |
| 涨停数量 | 20 | **10** | 进攻热度（易虚高，降权） |
| 连板高度 | 15 | **10** | 情绪标杆（极端值权重不宜过大） |
| **合计** | **100** | **100** | |

## 十五、职业交易员最终只看两件事

**第一**：风险是在扩大，还是在收缩。

**第二**：资金愿不愿意承担风险。

真正危险的时候，很多时候指数根本没怎么跌。

但：
- 炸板率上升
- 连板晋级下降
- 高位股补跌
- 成交额下降
- 主线瓦解
- 龙头独舞无跟风
- 龙虎榜游资集体隐身

这些东西，会提前告诉你：**市场已经开始退潮了。**

**复盘的终点不是看懂市场，是在看懂之后做出正确的仓位决策。**
